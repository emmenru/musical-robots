Proposed workflow for evaluation of musical robots:

  1. Classify the robot using existing taxonomies
  2. Specify the context (and goal, if there is such a thing) of the musical interaction
  3. Identify stakeholders
  4. Informed by steps 1–3, identify evaluation methods from the literature and adapt them, if necessary
